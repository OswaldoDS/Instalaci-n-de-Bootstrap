# Instrucciones

> Instala Bootstrap y ejecuta algunos componentes en tu replit.

## Instala Bootstrap con un enlace CDN
1. Ve a bootstrap.com [Bootstrap](https://getbootstrap.com/)
2. Desplázate hacia abajo y copia los enlaces CDN.
3. Instala esos enlaces en tu replit.

- Instala la etiqueta `link` dentro de la etiqueta `header`.

- Instala la etiqueta `script` debajo de la etiqueta `body`.

## Ejecuta tu primer botón
1. Abre un navegador con la documentación de Bootstrap [https://getbootstrap.com/docs/5.3/getting-started/introduction/](https://getbootstrap.com/docs/5.3/getting-started/introduction/) para añadir tus primeros componentes, como botones.

2. Explora el panel izquierdo para ver los componentes y, dentro de ellos, busca la sección de **botones**.
3. Copia el código de la **"clase básica"**.
4. Pega el código en el cuerpo de tu Replit.
5. Ejecuta el código para ver el botón.

## Botones de grupo en ejecución
1. Abre un navegador con la documentación de Bootstrap (https://getbootstrap.com/docs/5.3/getting-started/introduction/) para añadir tus segundos componentes, como los botones.

2. Desplázate por el panel izquierdo para ver los componentes y, debajo, busca **grupo de botones**.

3. Copia el código del **"ejemplo básico"**.
4. Pega el código en el cuerpo de tu Replit.
5. Ejecuta el código para ver el botón.

## Tarjeta en ejecución
1. Abre un navegador con la documentación de Bootstrap (https://getbootstrap.com/docs/5.3/getting-started/introduction/) para añadir tus terceros componentes, como la tarjeta.

2. Desplázate por el panel izquierdo para ver los componentes y, debajo, busca **tarjeta**.
3. Copia el código de **"ejemplo"**.
4. Pega el código en el cuerpo de tu Replit.
5. Ejecuta el código para ver el botón.
